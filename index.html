ループプレイリストメーカー for Spotify

3つの枠から1曲ずつ選び、指定時間ぎりぎりまで繰り返すプレイリストを作成します。

このツールはSpotify公式アプリではありません。入力内容はこのブラウザ内に保存され、サーバーには送信しません。Spotifyログインとプレイリスト作成にはSpotify Web APIを使用します。Client Secretは不要です。

Spotifyアプリ設定

各ユーザーが自分のSpotify Developer Dashboardで作成したアプリのClient IDを使います。Client Secretは不要です。

各ユーザー向けセットアップ手順
  1. Spotify Developer Dashboardを開き、Spotifyアカウントでログインします。
  2. Dashboardで「Create app」を押して、Spotifyアプリを作成します。アプリ名と説明は自由です。
  3. アプリ設定のRedirect URIに、以下のURLをそのまま追加します。最後の / まで一致させてください。
  4. 作成したアプリのClient IDをコピーし、このフォームのClient ID欄に貼り付けます。
  5. 曲URL、必要なら曲の長さ、プレイリスト名を入力して作成します。各枠は複数曲にできます。

長さ欄を空にするとSpotifyから自動取得を試します。Spotify側で拒否された場合は、3:45 のように手入力してください。

曲URLを入力するとジャケット画像、曲名、アーティスト名の表示を試します。長さはSpotifyログイン後に自動取得します。

長さ欄は入力後に欄を離れると、356 は 3:56、1000 は 10:00 に整形されます。

余白は、3時間を超えないように最後に空けておく秒数です。通常は 1 のままで問題ありません。

このツールはSpotify公式アプリではありません。入力内容はこのブラウザ内に保存され、サーバーには送信しません。

スマホでは、SafariやChromeの共有メニューからホーム画面に追加するとアプリのように起動できます。
